## Post Structure (Common)

Each post in `posts-blog-{a,b}.ts` follows the existing `Post` type:

```typescript
{
  id: string,            // e.g. 'a-002'
  title: string,         // keyword-optimized title
  category: string,      // '차량용품' | '일상'
  date: string,          // relative date string, e.g. '4일 전', '1주 전', '2주 전'
  author: string,        // persona author name
  authorProfileImage: string,  // reuse '/profile-a.jpg' or '/profile-b.jpg'
  headerImage: string,   // '/posts/{id}-header.jpg' (placeholder until images generated)
  tags: string[],        // include target keyword + related
  content: ContentBlock[],   // text + image blocks
  comments: Comment[],   // Q&A style, product link in reply
  isLiked: boolean,
  likeCount: number,     // randomize 5~20
  commentsEnabled: boolean,
  createdAt?: number,    // optional, use Date.now() - offset for realistic staggering
}
```

### Conventions

- **Content block IDs:** `{blogLetter}{postNumber}-{blockIndex}` (e.g. `a002-1`, `a002-2`, `b003-1`)
- **Comment IDs:** sequential starting at 1 within each post
- **Image paths:** `/posts/{id}-01.jpg`, `/posts/{id}-02.jpg` (placeholders)
- **Fields `commentsVisible` and `commentsOpen`:** optional, omit to use defaults (`true`)
- **Date staggering:** spread posts across 1~14 days, newer posts with smaller numbers. Use `N일 전` format only (parser does not support `주 전`)

## Content Guidelines

1. **No product name in body.** Describe the product by experience ("태양열이라 선 연결 없고", "번호판이랑 겸용") but never name it directly.
2. **Product link only in comments.** Always via Q&A pattern: someone asks → author replies with cafe24 link.
3. **Each post 5~7 content blocks.** Mix of text (4~5) and image placeholders (2~3).
4. **Tags include target keyword.** Plus 2~3 related tags for discovery.
5. **Like counts vary.** 5~20 range, randomized to look natural.
6. **Comment timestamps staggered.** Realistic gaps (hours to days after post).

## Image Generation Strategy

- Tool: Google AI (Nano Banana 2 model)
- Style: iPhone 15 photo, natural lighting, no filters, realistic Korean settings
- Per post: 2~3 images (header + 1~2 in-body)
